//CREATE
Route = /register
In this route I am creating a user record in my database(MongoDB) and saving the encrypted password.

//READ
Route = /readUser
In this route I read the user information by using username as the primary key.

//UPDATE
Route = /updateUser
In this route, I am updating the user's name in the database by using username as the primary key.

//DELETE
Route = /deleteUser
In this route, I am deleting the user's record from my database using username as the primary key.
